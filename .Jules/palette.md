## PALETTE'S JOURNAL

## 2025-02-28 - Search Empty State & Keyboard Accessibility
**Learning:** Empty search states without a clear call-to-action leave users frustrated, while movie list cards lacked visual indicators for keyboard navigation.
**Action:** Added a friendly "No movies found" empty state with a "Clear Search" button to reset the query. Improved keyboard accessibility by adding focus rings (`focus:ring-2 focus:ring-sky-600 focus:ring-offset-2`) to movie cards and the search submit button. Also improved screen reader support by adding an explicitly linked `sr-only` label to the search form input.
